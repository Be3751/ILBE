# ILBe - Items Left Behind
## 概要
研究室にある放置物品を管理するためのWebアプリケーションを制作しました。  
研究室に置き忘れられた物品を登録することで自動で廃棄期限を設け、不要物品を一覧表示します。  

## 機能
本アプリの機能は以下です。
- 物品の登録
- 登録時のLINE通知
- 物品の一覧表示
- 物品一覧の表示切り替え
- 所持品申請

## 実装
フロントエンド：Bootstrap  
バックエンド：Django
API：LINE Notify
